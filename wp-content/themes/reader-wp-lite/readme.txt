== Description ==

Reader WP Lite is a Clean and Modern Designed Personal Blogging WordPress Theme. It's suitable for Bloggers, Photographers, Personal Portfolio websites. Clean and Stunning designed WordPress Theme. Comes with Customizer Support, Widgets Ready, Post Formats Supports, Background Image Posts Options. Reader WP Lite is Translation Ready Theme.
Our Support: http://prowptheme.com/support/

== Installation ==
        
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the wp-reader-lite.zip file. Click Install Now.
3. Click Activate to use your new theme right away.
